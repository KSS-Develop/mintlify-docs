## Core Capabilities

| Module | Description |
|--------|-------------|
| **Tax Documents (DTE)** | Emit Facturas Exentas (T34), Notas de Debito (T56), Notas de Credito (T61) |
| **Boletas & Dispatch Guides** | Boletas electronicas and Guias de Despacho |
| **Received Documents** | Sync and manage DTEs received from suppliers via SII |
| **Contacts & Products** | Full contact directory and product catalog with stock management |
| **Accounting** | Chart of accounts, journal entries, general ledger, balance sheet, income statement |
| **Inventory** | Multi-warehouse, transfers, Kardex, valuation |
| **Financial Analysis** | Cash flow projections, budgets, profitability, collections aging |
| **Bank Reconciliation** | Automatic and manual bank statement matching |
| **Quotes & Purchase Orders** | Full quoting lifecycle with DTE conversion |
| **Tax & Books** | Sales/purchases books, F29, VAT detail, tax regime simulation |
| **MercadoPago** | Payment integration with automatic sync |
