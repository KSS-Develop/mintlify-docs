## Step 1: Get your credentials

Log in with your email and password to obtain a JWT access token.

```bash
curl -X POST https://backend-production-244de.up.railway.app/api/v1/usuarios/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your-email@company.cl",
    "password": "your-password"
  }'
```

Response:

```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer",
  "user": {
    "id": 1,
    "email": "your-email@company.cl",
    "companies": [
      { "id": 1, "rut": "76123456-7", "razon_social": "Mi Empresa SpA" }
    ]
  }
}
```

<Note>
If your user has access to multiple companies, the token is scoped to the first company. Use the [Switch Company](/api-reference/endpoint/usuarios/post-usuarios-switch-company) endpoint to change.
</Note>

## Step 2: Use the token

Include the token as a Bearer header in all subsequent requests.

```bash
curl https://backend-production-244de.up.railway.app/api/v1/dashboard/stats \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```

## Step 3: Emit your first DTE

Emit a Factura Exenta Electronica (type 34):

```bash
curl -X POST https://backend-production-244de.up.railway.app/api/v1/dte/emitir \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..." \
  -H "Content-Type: application/json" \
  -d '{
    "tipo_dte": 34,
    "receptor": {
      "rut": "66666666-6",
      "razon_social": "Cliente Ejemplo SpA",
      "giro": "Comercio al por menor",
      "direccion": "Av. Providencia 1234",
      "comuna": "Providencia"
    },
    "detalle": [
      {
        "nombre": "Servicio de consultoria",
        "cantidad": 1,
        "precio": 100000,
        "exento": true
      }
    ]
  }'
```

Response:

```json
{
  "tipo_dte": 34,
  "folio": 123,
  "fecha": "2025-01-15",
  "monto_total": 100000,
  "estado_sii": "pending",
  "track_id": "987654321"
}
```

## Step 4: Check DTE status

The DTE is signed and uploaded to SII asynchronously. Check its status:

```bash
curl https://backend-production-244de.up.railway.app/api/v1/dte/34/123/estado-avanzado \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```
